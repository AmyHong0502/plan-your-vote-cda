Mapbox - Initial Setup

  1. Go to Mapbox website
  2. Create an account
  3. Login
  4. The screen will look like this:
  5. Copy the default token. It will be used in your CMS.
  6. Click Create a token button.
  7. Create a new token. Don't forget to add URL restrictions.
  8. Copy the new token. It will be used in your CDA.
